Few Words About The POS Orders Hourly Report:

This module gives you an option to get the POS Orders report by hours, means In a particular hour which orders are placed.

Work Flow: How the "POS Orders Hourly Report" works
Here in the POS pivot report option you get the extra option for row or column:
  • i.e. Order in Hour
Set Column:
  • Select Order Date as column and select day a interval for date
Set Row:
  • Select Order in Hour as row
Report: Pivot View
