POS Report

Key Features:

  • Allow users to print X and Z report.
  • User can see the Summary of sales, Top Selling product, Category wise sales.
  • X report will shows the data of open session, while Z report will shows the data of closed session.

Overview

  • For print X Report, Go to menu Point of Sale > Reporting > X Report. Select the search criteria and click on print button.
  • X Report PDF.
  • For print Z Report, Go to menu Point of Sale > Reporting > Z Report. Select the search criteria and click on print button.
  • Z Report PDF.
